# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-11

### Added
- Initial release of Jeenee UI Builder node
- Support for all Jeenee DSL components:
  - Buttons (callback & URL)
  - Select Menu (single/multi)
  - Checkboxes
  - Text Input
  - File Upload (single/multiple files)
  - Rich Choices
  - Image Choices
  - Notes and Progress indicators
  - Images
- Visual GUI configuration for all components
- TypeScript support with full type definitions
- Comprehensive documentation and examples

### Features
- Build Telegram bot UIs visually through n8n
- Support for complex interactive components
- File upload with S3 integration
- Multiple file uploads with customizable limits
- Rich content blocks for advanced UX
- Session management and metadata support
